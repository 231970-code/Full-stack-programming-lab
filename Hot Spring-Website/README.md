# HotSpring Portable Spas

## Full Stack Programming Assignment 01

A fully responsive e-commerce website built with HTML5, CSS3, Bootstrap 5, JavaScript, and jQuery.

---

## Project Structure

```
Task/
├── index.html                 # Home page
├── css/
│   └── style.css             # Main stylesheet with custom styles
├── js/
│   └── main.js               # JavaScript with jQuery validation & animations
├── images/                    # Image assets folder
└── pages/
    ├── about.html            # About Us page
    ├── category.html         # Products/Category listing
    ├── contact.html          # Contact Us page with form
    ├── login.html            # User login page
    ├── register.html         # User registration page
    ├── forgot-password.html  # Password reset page
    ├── product.html          # Product details page
    ├── cart.html             # Shopping cart page
    ├── payment.html          # Payment/checkout page
    ├── terms.html            # Terms & Conditions page
    └── account/
        ├── my-account.html      # Account dashboard
        ├── edit-account.html    # Edit profile page
        ├── edit-billing.html    # Edit billing address
        ├── edit-shipping.html   # Edit shipping address
        ├── order-summary.html   # Order history
        └── order-details.html   # Individual order details
```

## Form Validations

### Registration Form

- First Name & Last Name (required)
- Email Address (valid format)
- Phone Number (10+ digits)
- Password (8+ chars, uppercase, lowercase, number)
- Confirm Password (must match)

### Login Form

- Email Address (valid format)
- Password (required)

### Contact Form

- Name (2+ characters)
- Email (valid format)
- Message (10+ characters)

### Payment Form

- Cardholder Name (required)
- Card Number (16 digits, auto-formatted)
- Expiry Date (MM/YY format)
- CVV (3-4 digits)
- Billing & Shipping addresses (complete fields)
